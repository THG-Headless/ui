## Rounding

Corner radii can be set using the following parameters:

- `--default-rounded-btn` sets the corner radius of buttons
- `--default-rounded-box` sets the corner radius of boxes *(e.g. cards)*
- `--default-site-corners` sets the corner radius of top level components *(like containers)*
