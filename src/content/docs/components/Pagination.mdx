## Disabled

The `Pagination` component supports a disabled state for the page selector, by applying a `disabled` attribute. 

Note: If the "select" is disabled it is best to either hide or disable the prev/next buttons. 

<ShowComponent filePath="./src/components/pagination/Pagination_Disabled.html">
  <PaginationDisabled />
</ShowComponent>

## CSS Variables

The `Pagination` component uses the following CSS variables for customization:

| Variable                              | Default Value       | Description                                   |
|---------------------------------------|---------------------|-----------------------------------------------|
| `--pagination-select-padding-top`     | `0.25rem`           | Top padding for the page selector.           |
| `--pagination-select-padding-bottom`  | `0.25rem`           | Bottom padding for the page selector.        |
| `--pagination-select-padding-left`    | `0.75rem`           | Left padding for the page selector.          |
| `--pagination-select-padding-right`   | `2rem`              | Right padding for the page selector.         |
| `--pagination-select-border-width`    | `1px`               | Border width for the page selector.          |
| `--pagination-select-border-width-disabled` | `2px`         | Border width for the disabled page selector. |
| `--pagination-select-background-size` | `1rem 1rem`         | Background size for the dropdown arrow.      |
| `--pagination-select-background-position` | `right 0.5rem center` | Background position for the dropdown arrow. |



### Accessibility
#### Implemented Accessibility Features

- [ ] Includes proper `aria-label` attributes for navigation buttons
- [ ] Provides visible focus indicators for keyboard navigation
- [ ] Use of semantic HTML elements `<nav>` and `<select>` 
- [ ] Tab key navigates between pagination controls
- [ ] Arrow keys change values in the select dropdown
- [ ] Disabled controls are properly excluded from tab order

#### Accessibility- Implementation Notes

We recommend removing buttons that aren't clickable with JavaScript, for the first and last pages, e.g:
