# Basic Usage

The basic markup for a pagination component comes with sensible defaults for `:hover`, `:focus`, `:active` pseudoclasses. 

<ShowComponent filePath="./src/components/pagination/Pagination.html">
  <Pagination />
</ShowComponent>


# Disabled

Use the `disabled` directive to apply disabled styles to the next/prev buttons or to the select

<ShowComponent filePath="./src/components/pagination/PaginationNextDisabled.html">
  <PaginationNextDisabled />
</ShowComponent>

You can similarly disable the 'select':

<ShowComponent filePath="./src/components/pagination/PaginationSelectDisabled.html">
  <PaginationSelectDisabled />
</ShowComponent>
